# :computer: How to run

```bash
# Clone Repository
$ git clone https://github.com/GabriellCdeC/teste-gobetti-rh
```

```bash
# Install Dependencies
$ yarn or npm install

# Run Aplication
$ yarn start or npm start
```
Go to http://localhost:3000/ to see the result.
